# BB877 Utility Bot

An all-in-one Telegram utility bot.

## Features

- Fancy Text
- Password Generator
- Random Number
- Coin Flip
- Dice
- Uppercase
- Lowercase
- Reverse Text
- Word Counter
- Password Strength
- SHA256
- Base64
- UUID
- Timestamp
- Random Quotes
- Emoji
- Guess Number Game

Built with Python and python-telegram-bot.

Hosted on Railway.
