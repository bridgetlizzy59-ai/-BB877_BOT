# BB877 Utility Bot

A multifunction Telegram utility bot built with:

- Python 3.12
- python-telegram-bot 21.10
- Async architecture
- Inline keyboard navigation
- Railway deployment

## Features

- Fancy Text (20+ Unicode styles)
- Password Generator
- Password Strength Checker
- SHA-256
- Base64 Encode
- Base64 Decode
- UUID Generator
- Unix Timestamp Converter
- Uppercase
- Lowercase
- Reverse Text
- Word Counter
- Random Number Generator
- Coin Flip
- Dice Roller
- Random Emoji
- Random Quote
- Guess Number Game
- Help
- About

## Railway Deployment

Create a Railway project.

Add environment variable:

BOT_TOKEN

Deploy.

## Local

```bash
python -m venv .venv

source .venv/bin/activate
```

Install:

```bash
pip install -r requirements.txt
```

Run:

```bash
python bot.py
```
